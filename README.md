# 🧠 study-notes | 人工智能相关技术的学习笔记与实验记录

作为一名人工智能方向的学习者，我创建这个仓库来：

-> 记录我对人工智能、深度学习、机器学习等相关技术的理解与整理

-> 拆解经典算法、模型结构与核心思想，结合代码加深认识

-> 验证我对每个模块、每个函数、每个参数的理解

-> 收集并解答学习过程中的疑问、错误与踩坑经验

-> 保存实验过程、实践结果和阶段性的学习总结

这不是一份“标准答案”，而是一段持续探索、不断修正的真实学习记录。

我希望通过整理笔记，把零散的知识慢慢连接成体系；  
也希望通过动手实验，让理解不只停留在表面。

目前学习内容主要包括：

- 人工智能基础
- 机器学习
- 深度学习
- 神经网络相关原理
- 经典模型与代码阅读
- 实验记录与问题总结

欢迎一起交流、讨论、指正和完善。
